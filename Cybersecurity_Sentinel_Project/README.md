Cybersecurity Hands-On Project: Build a SOC with Azure Sentinel

This project demonstrates how to build a Security Operations Center (SOC) using Azure Sentinel to monitor and respond to real-world attacks.

Key Features:

Collect and analyze security logs from Azure resources

Write KQL queries to detect suspicious activity

Visualize threats on a real-world map using Azure Workbooks

Gain hands-on experience with cloud security monitoring and incident response
