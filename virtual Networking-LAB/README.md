
# Azure Virtual Network Project
This project demonstrates how to create a secure virtual network in Azure with multiple subnets, NSG rules, and connectivity testing.

## Steps Performed
1. Created a Resource Group and Virtual Network.
2. Added Subnets for different workloads.
3. Configured NSG rules.
4. Tested DNS and connectivity using `nslookup` and `ping`.
5. Deployed via ARM Template.
